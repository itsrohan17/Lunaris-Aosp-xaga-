# Changelog - Lunaris AOSP (xaga)

## [3.8] - 2026-03-06
### Added & Improved
- **Design:** Redesigned recent overview with improved L3 animations and accent color privacy indicators.
- **Customization:** Added Clock faces, Gradients, and Cutout progress ring (notch ring).
- **Features:** Support for daily/weekly data usage cycles and SSID labels on tiles.
- **System:** Merged March Security Patch (r4) and switched to Gsans Flex as default font.

### Fixed
- Fixed Battery Widget stuck/loading issue.
- Fixed settings crash after resetting default fonts.
- Resolved torch and WiFi lockscreen widget issues.

---

## [3.7] - 2026-02-17
### Added
- **Launcher:** Added categorized folders (Lawnchair Caddy) and Volume panel styles.
- **System:** Added Island notifications, Ambient display (CPU temp/User name), and Shake gestures.
- **Device Specific (xaga):** Added Lunaris Dolby, Xiaomi touch for gaming, and ViperFX.

### Fixed & Optimized
- Fixed PixelProp and Integrity issues.
- Optimized ServiceManager priority and multitasking performance.
